# jsdvi

## これは何

JavaScriptでdviプレビューアを書いてみるというネタ

## 何が嬉しいの

ブラウザからコピペできるとか

## 制限

* TFMを読んでないのでメトリクスが適当。
* 位置を揃えることよりコピペ可能であることを優先。
* 後述のwebフォント以外のフォントが来ると当然ながら文字化けする。
